Git Github

CONFIGURACIONES:


** git config --global user.name"[name]"
** git config --global user.email "[email]"
** git config --lit //es para comprobar las configuraciones que tenemos


CREAR:


** git clone[url] //crear un repositorio
** git init //crear un repositorio


VERIFICAR:


** git status //estado actual del proyecto
** git log //para el historial de los commits
** git diff //para ver los cambios en el archivo
** git branch //para ver los branch que tenemos dentro del proyecto


AGREGAR:


** git add. //agregar todos los archivos
** git add [archivo] //agregar un archivo en especifico


MOVER:


** git commit -m "[mensaje]" //para mover los archivos de un area a otra


ENVIAR O ACTUALIZAR:


** git push

BAJAR O ACTUALIZAR DESDE EL REMOTO:


** git pull

CAMBIAR:


** git checkout -b [branch-name] //crear una rama nueva
** git merge //para mezclar las ramas
** git switch //para salir de una rama a otra
** git restore --source[hash][archivo] //es para restaurar el estado a un commit en especifico
